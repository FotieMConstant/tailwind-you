# Introduction
 
Tailwind You is a beautiful component library that combines the power of [Tailwind CSS](https://tailwindcss.com) with Google's new [Material You](https://m3.material.io/)(Material 3) design principles for creating beautiful and personalized user interfaces for the web.
With a wide range of pre-built components, Tailwind You makes it easy to create beautiful and functional user interfaces that are tailored to your users' preferences.

## Documentation
In this documentation, you will learn how to use **@tailwind-you/react** and **@tailwind-you/vue** components and how you can integrate it in your own projects.
This library is meant to help you biuld your frontend based on **Material You** design guidlines using Tailwind CSS - [Installing Tailwind CSS](https://tailwindcss.com/docs/installation)

Tailwind You uses Tailwind CSS classes, thus you must install Tailwind CSS on your project in order for it to work properly. Preferably, **Tailwindcss 3.x.x**
